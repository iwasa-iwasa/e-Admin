# カレンダーとメモの連携機能 - マイグレーション手順

## 実行するコマンド

```bash
php artisan migrate
```

このコマンドで以下のマイグレーションが実行されます：
- `2025_01_20_000000_add_linked_event_id_to_shared_notes_table.php`

## 変更内容

### データベース
- `shared_notes`テーブルに`linked_event_id`カラムを追加
- カレンダーイベントと共有メモを確実に連携

### 機能
1. **カレンダー → メモの連携**
   - カレンダーイベント作成時、詳細が入力されていれば共有メモを自動作成
   - ジャンル（会議、業務、来客、出張、休暇）が色として共有メモに反映

2. **カレンダー編集 → メモ更新**
   - カレンダーイベントを編集すると、連携された共有メモも自動更新
   - タイトル、内容、ジャンル、重要度、期限、参加者が同期

3. **メモ編集 → カレンダー更新**
   - 連携された共有メモを編集すると、カレンダーイベントも自動更新
   - タイトル、内容、ジャンル、重要度、期限、参加者が同期

## 注意事項
- 既存のデータには影響しません（`linked_event_id`はnullable）
- 新規作成されるイベント・メモから連携が有効になります
