# 運用ルール追記箇所

## ヘルプマーク（HelpCircle）への運用ルール追記

### 1. SharedCalendar.vue（カレンダー画面）
**追記内容**：
```
【他部署メンバーを招待する場合】
- 「他部署のメンバーも追加」にチェックを入れて選択してください
- 他部署メンバーを招待する予定には、共有メモをリンクすることを推奨します
- 共有メモに会議の目的、議題、資料を記載すると、参加者が事前に内容を確認できます

【予定の所属について】
- 予定は選択したカレンダーに所属します
- 例：営業部カレンダーで作成 → 営業部の予定として表示されます
- 他部署メンバーには「他部署の予定に招待されました」と通知されます
```

**追記箇所**：
- カレンダー作成ダイアログの参加メンバー選択部分
- 既存のHelpCircleツールチップに追記

---

### 2. Notes.vue（共有メモ専用画面）
**追記内容**：
```
【共有メモの活用方法】
- カレンダー予定にリンクすることで、詳細情報を共有できます
- 特に他部署メンバーが参加する予定には、共有メモのリンクを推奨します
- 会議の議題、資料、事前準備事項などを記載してください

【公開範囲の設定】
- 全社公開：全社員が閲覧可能
- 自部署のみ：自分の部署のメンバーのみ閲覧可能
- 一部ユーザーのみ：選択したメンバーのみ閲覧可能
```

**追記箇所**：
- 共有メモ作成・編集ダイアログ
- 既存のHelpCircleツールチップに追記

---

### 3. DashboardHeader.vue（通知センター）
**追記内容**：
```
【通知の見方】
- 🏷️ バッジ：予定が所属するカレンダー（全社/部署名）を表示
- ⚠️ 他部署の予定に招待されました：自分の部署以外の予定に参加する場合
- 📝 共有メモあり：詳細情報が共有メモにリンクされています

【フィルターの使い方】
- 自分のみ：自分が作成または参加する予定のみ
- 自部署のみ：自部署で作成または参加する予定のみ
- 全て：作成者または参加者として関わる全ての予定
```

**追記箇所**：
- 通知センターのヘッダー部分
- 既存のHelpCircleツールチップに追記

---

### 4. CreateEventDialog.vue（カレンダー作成ダイアログ）
**新規追加**：
```
【参加メンバーの選択】
- デフォルト：選択したカレンダーの部署メンバーのみ表示
- 「他部署のメンバーも追加」にチェック：全メンバーから選択可能

【他部署メンバーを招待する場合の推奨事項】
✓ 共有メモをリンクして詳細情報を共有
✓ 会議の目的や議題を明記
✓ 必要な資料を共有メモに添付

【予定の通知】
- 参加者全員に通知が送信されます
- 他部署メンバーには「他部署の予定に招待されました」と表示されます
```

**追記箇所**：
- 参加メンバー選択エリアの近くにHelpCircleアイコンを新規追加

## ツールチップの実装方法

既存のHelpCircleアイコンは以下のような実装になっています：

```vue
<TooltipProvider>
  <Tooltip>
    <TooltipTrigger as-child>
      <button class="text-gray-400 hover:text-gray-600">
        <HelpCircle class="h-5 w-5" />
      </button>
    </TooltipTrigger>
    <TooltipContent class="max-w-xs">
      <p class="text-sm">
        <!-- ここに運用ルールを追記 -->
      </p>
    </TooltipContent>
  </Tooltip>
</TooltipProvider>
```

**注意点**：
- ツールチップの幅は `max-w-xs` または `max-w-sm` で調整
- 長文の場合は改行を適切に入れる
- 箇条書きで見やすく整理
