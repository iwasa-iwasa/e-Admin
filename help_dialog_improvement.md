# ヘルプダイアログUI改善方針

## 問題点
- 図解（ボタンやアイコン）とテキスト説明が横並びで配置されており、狭いスペースで見づらい
- 固定幅（w-32など）で図解エリアを確保しているため、レスポンシブ性に欠ける

## 改善方針
各機能説明カードを以下の構造に統一:

```
┌─────────────────────────────────────┐
│ [アイコン] 機能名                    │
├─────────────────────────────────────┤
│ 説明テキスト                         │
└─────────────────────────────────────┘
```

- アイコン/ボタンとタイトルを1行にまとめる
- 説明文は下部に配置
- 図解が必要な場合は、説明文の前に独立した行として配置
